MagniPOS is a Free Point of sale(POS) system for WooCommerce. MagniPOS provides you APPS for Android, iOS, Windows and MacOS. It utilizes 
WooCommerce and WordPress Rest API to connect with the APPS.

== Description ==
MagniPOS is a simple, user friendly Point of Sale system built on top of WooCommerce. It's built specifically for physical stores to easily sell both online and offline using WooCommerce without having to worry about the stock management. 
This is an one-stop solution that takes care of everything from order management to inventory, help you to streamline & capable of handling many of the operational tasks such as order status update, billing, invoicing and set your business goals.

MagniPOS automatically integrates with your existing WooCommerce store within 2 minutes and becomes production ready in 5 minutes.
You can easily connect any POS printer and barcode scanning machine with MagniPOS to make your physical billing more powerful.

* Automatically sync inventory between online and offline store.
* Automatically sync all WooCommerce products.

== Installation ==

1. Download the plugin and unzip it.
2. Upload/Copy the folder *magni-pos* to your /wp-content/plugins/ folder.
3. Activate the plugin from your WordPress admin panel.
4. Installation finished.

== Frequently Asked Questions ==

= Does this plugin has a settings page? =
Yes, it has. You can activate your lifetime free trial from the settings with single click.

= Does this plugin work with multi-site? =
Yes, this plugin is compatible with multi-site.

= Does this plugin has any dependency? =
Yes, it needs WooCommerce to be installed and activated in order to work.

= How do I connect my Store to MagniPOS? =
Just install the MagniPos Plugin and hit connect on the setting and you will be connected. After that you just need to enter your details on the MagniPOS APP to access your inventory and billing dashboard.
